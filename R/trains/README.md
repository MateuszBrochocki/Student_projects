# Interactive visualization of passenger rail transport in Poland

## Description

The interactive application is divided into three parts. The first section describes the railway situation by province. The second covers railway lines, and the third covers delays. Each section includes interactive charts enabling in-depth data exploration. The project was made in RStudio.
